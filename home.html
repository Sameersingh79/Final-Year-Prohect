Prognostication Of Diseases

Disease Prediction using data science

Prognostication of Diseases" system is based on prevised modeling which predicts the disease of the user on the basis of the symptoms thst user provides as an input to the systems. This system analyses the symptoms provided by the user as an input and gives the probability of the disease as an output.

The Project aims to reduce the burden of the people by providing all the information they need to know and detect the disease early. Disease Prediction is done by Naive Bayes Classifier. Naive Bayes Classifier calculates the probability of the disease.
